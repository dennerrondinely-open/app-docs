## Observações

- Alguns contratos usam `queryParameters` para filtros e identificadores.
- Alguns endpoints usam path dinâmico composto em runtime (ex.: contratos de documento em `product`).
- Downloads usam `downloadFile` ou `ResponseType.bytes`, portanto não retornam JSON.
