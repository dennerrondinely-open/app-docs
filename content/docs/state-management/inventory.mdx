## Observação de manutenção

Como o projeto possui muitos blocs, a documentação privilegia:

- blocs estruturais
- blocs de jornada crítica
- nomes reais de `event` e `state`
- localização dos arquivos para leitura aprofundada

A recomendação é manter esta página atualizada sempre que um novo fluxo for criado ou quando um `Bloc` mudar de responsabilidade.
