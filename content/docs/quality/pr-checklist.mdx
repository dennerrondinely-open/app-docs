---
title: Checklist de PR
description: Padrão mínimo para manter qualidade e previsibilidade de entrega.
---

## Antes de abrir PR

- [ ] Build local da flavor alterada
- [ ] Testes unitários da feature
- [ ] Atualização de documentação (rotas, endpoints, modelos)
- [ ] Sem credenciais, segredos ou dados sensíveis no diff

## Antes do merge

- [ ] Revisão técnica aprovada
- [ ] Checklist de produto validado
- [ ] Sem regressão em fluxos críticos
- [ ] Plano de rollback definido para mudanças sensíveis
