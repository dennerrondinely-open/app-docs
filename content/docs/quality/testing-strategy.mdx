## Pirâmide de testes

- Testes unitários para `models`, `repositories` e `utils`
- Testes de `bloc` para transições de estado
- Testes de integração para fluxos críticos (quando aplicável)

## Ferramentas já usadas

- `flutter_test`
- `bloc_test`
- `mocktail`

## Comandos

```bash
fvm flutter test
fvm flutter test --coverage
fvm flutter analyze
```
